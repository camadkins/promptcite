# Code of Conduct

PromptCite adopts the **Contributor Covenant** version 2.1 as its Code of
Conduct, verbatim. The canonical text is published by the Contributor
Covenant project at:

- **Full text:** https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- **Translations:** https://www.contributor-covenant.org/translations/

By participating in this project — opening issues, submitting pull
requests, joining discussions, or interacting with any other
contributor in any project space — you agree to abide by that document
in full.

## Scope

This Code applies to all project spaces: this GitHub repository
(issues, pull requests, discussions, code review, commit messages),
any official communication channels the project may add later, and
any space where an individual is representing the project in public.

## Enforcement

Reports of unacceptable behavior should be sent to the maintainer via:

- **Email:** `cam@camadkins.com`

The maintainer will review and investigate all complaints and respond
in a way that is deemed appropriate to the circumstances. All
complaints are kept confidential. The maintainer is obligated to
maintain confidentiality with regard to the reporter of any incident.

Maintainers who do not follow or enforce the Code in good faith may
face temporary or permanent repercussions as determined by other
project leadership.
